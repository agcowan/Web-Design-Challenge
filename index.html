Summary: Latitude comparisons

To learn about how APIs work, we undertook a project to learn about weather conditions at different latitudes across the globe. We sought to find and plot the relationships among weather data and latitudes. This project was accomplished using the OpenWeatherMap API to pull data from over 500 cities.


From the data, we used MatPlotLib to construct the plots that compared latitude to cloudiness, humidity, maximum temperature, and wind speed. The API calls took in latitude and longitudes and returned the closest city.
